18 /*
19 The Transform structure holds information about the three main transformations for a 3D rigidbody: position, rotation, and scale.
20 Note that position is an affine transformation, so this structure is not equivalent to the Basis3D structure, which only holds linear transformations.
21 Also, the basis vectors that the Transform produces are always orthogonal (perpendicular to one another), since they're just rotated and scaled.
22 */
23
